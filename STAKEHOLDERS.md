# STAKEHOLDERS — 誰が関わり、どう決めるか

AI.LandBase の関与者と意思決定の構造を定める。チームは **DAO 的（分散的・合意ベース）** な運営を志向する点が特徴。

## ガバナンスの方針

中央集権的なトップダウンではなく、**DAO 的な分散運営**を基本姿勢とする。

- 役割は固定的なヒエラルキーではなく、貢献と専門性に応じて流動的に持つ
- 重要な意思決定は、限られた決裁者の独断ではなく合意形成を通す
- 透明性（Values の 7 番）をガバナンスにも適用し、決定の過程を見える形にする

> 📝 要記入（公開情報に無い・本人確認が必要）:
> - 具体的なメンバー構成と各自の役割
> - 「DAO 的」の実装度合い — トークン/オンチェーンを伴う本来の DAO か、比喩としての分散運営か
> - 意思決定プロセスの具体（何を・誰が・どう決めるか、合意の取り方）
> - 代表者・法的な責任者（登記上の代表）

## 関与者の区分（枠）

| 区分 | 役割 | 備考 |
|---|---|---|
| コアチーム | 事業の意思決定と実行 | 構成は要記入 |
| 顧客（観光事業者） | 価値の受け手であり共創者 | [AUDIENCE.md](./AUDIENCE.md) |
| 地域・連携先 | エコシステム構築の協働相手 | Values 5「協働とエコシステム構築」 |

## 補足

「DAO 的」という語は多義的なため、上記の要記入が埋まるまでは**比喩としての分散運営**の意味で用いる。本来の DAO（トークン・オンチェーン統治）を指す場合は、その旨と仕組みを本 doc に明記する。

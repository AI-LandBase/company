# 株式会社 AI.LandBase（アイランドベース）

> 沖縄の観光業に、AI という伴走者を。

沖縄県北部・今帰仁村を拠点に、観光業のための AI 経営支援を提供する会社です。
このリポジトリは、会社の **Mission・Vision・事業戦略・ブランド** を公開ドキュメントとして残す場所です。

## Mission

沖縄県北部の観光業事業者（ホテル・飲食店・ツアー会社など）に AI テクノロジーの力を届け、データドリブンな経営への転換を支援することで、持続可能な観光産業の発展に貢献する。

## Vision

2030 年までに、沖縄県北部の全観光業事業者が AI を活用した戦略的経営を実践し、データと人間の知恵が融合した新しい観光エコシステムのモデル地域となる。

## 解決する課題

1. **人手不足** — 限られた人手で運営し続けるための省力化と伴走
2. **データ活用の停滞** — 経営判断に使えていないデータを意思決定に変える
3. **季節による売上の波** — 繁閑差をならし、年間を通じた安定経営へ

## 提供価値（概要）

統合された AI ツール群と「伴走型」の支援で、観光事業者の経営を継続的に支えます。
詳しい競争戦略・ビジネスモデルは [STRATEGY.md](./STRATEGY.md) を参照してください。

# AUDIENCE — 誰に向けるか

AI.LandBase が向き合う相手を、**顧客セグメント**（事業の相手）と**このリポジトリの読み手**（発信の相手）に分けて定める。

## 顧客セグメント（事業の相手）

### 一次顧客: 沖縄県北部の観光業事業者
- ホテル・宿泊施設、飲食店、ツアー会社など
- 限られた人手で運営し、データ活用や季節変動に課題を抱える
- 大手向けの汎用 SaaS では距離が遠く、相談しやすい伴走相手を求めている

> 📝 要記入: 規模感（客室数・従業員数のレンジ）、IT リテラシー、典型的なペインの具体像。導入事例が出たらペルソナを実データで肉付けする。

## このリポジトリの読み手（発信の相手）

このリポは公開ドキュメント。主に次の読み手を想定する。

| 読み手 | 何を知りたいか | どの doc を見るか |
|---|---|---|
| 採用候補者 | どんな会社か、何を目指すか、どんな価値観か | README / BRIEF / BRAND |
| 外部パートナー・連携先 | 何の会社で、どう組めるか | README / STRATEGY |
| 投資家・行政等 | Mission/Vision、事業の筋、地域へのインパクト | BRIEF / STRATEGY |
| 将来の自分・チーム | 当時何をどう決めたか | 全 doc |

## 何を出し、何を出さないか

- **出す**: Mission/Vision/Values、事業の考え方、ブランドの姿勢
- **出さない**: 料金の生数字（→ 公式サイト）、顧客の固有情報（許諾なしの実績）、内部の機密
